1 Starting Emacs

1.1 Options to run Emacs

  • [X] WSL in terminal (I use cmder) with bash (Ubuntu)
  • [X] XServ graphics
  • [X] VSCode extension (no agenda view)
  • [ ] Windows Emacs MSI executable (bad Russian font spacing)

Console vs teminal vs shell.

1.2 Basic pipeline

  • Install emacs on WSL with sudo apt-get install emacs
  • Launch emacs with emacs -nw <FILENAME.org> command
  • Open a file with F10 - File menu
  • Edit file
  • Save file C-x C-s
  • Exit Emacs C-x C-c

1.3 Improvements and changes

1.3.1 DONE Configure terminal to avoid key conflicts

Must configure terminal to resolve key binding conflicts like Ctrl-C.

1.3.2 DONE Set org-support-shift-select and CUA option

  • Selection with Shift is already built-in part of Emacs, but not org-mode
  • Start with 'M-x customize' to find options about Shift selection
  • Set CUA (Ctrl-Z/X/C/V) as part of F10 menu
  • Save options
  • See also org-disputed-keys in Orgmode conflicts.

1.3.3 DONE Where is the config?

Use just one, any of them:

  • ~/.emacs is a config file
  • ~/.emacs.d is a directory with init.el

3 Using orgmode

3.1 <TAB> is all you need

  • TAB shows/hides headers (quite powerful!)
  • Shift-TAB opens all headers

3.2 Getting around headers

  • Alt + left or right changes header level
  • Alt + up or down moves lines around
  • Shift - arrow:
    • changes list numbering style
    • cycles TODO-DONE in header
    • selects in CUA mode

3.7 Clocking

* Clock this!

  • Start: C-c C-x C-i
  • End: C-c C-x C-o

It is easy to “clock-in” to a particular task by positioning point within an item and typing C-c C-x C-i and clocking out with C-c C-x C-o.

https://www.adventuresinwhy.com/post/org-mode-timekeeping/

5 Reference

5.1 Concepts

a screen that represents a file or Emacs own output
frame
is a new window for the whole program
modeline
a line at the bottom of a screen with something like -UUU(DOS)**--F1
window
is a windows inside editor

8 Appendix. Key binding cheatsheet

Command Action
M-x <command> Run command by name
M-x shell Run Shell
F10 Upper menu
C-s Advanced search
C-c C-c Toggle or recalculate
C-k C-y Kill and undo line
C-g Kill, stop or exit
ESC-ESC-ESC Exit (hopefully)
q Exit (sometimes)
C-c C-e h h Create HTML
Not Emacs:  
Fn-Esc Lock Fn key (Lenovo)

  • revert-buffer (in File menu)
  • customize
  • shell
